# Three-Lane Swamp Valley — authored semantic map pack

Imported from ChatGPT layer export (2026-07-10).

## Layout

- `manifest.json` — layer index, color palettes, import rules
- `verification.json` — upstream QA report (`fix_status: PASS`)
- `source/` — native **1254×1254** PNG layers (authoring export)
- `baked/350/` — resampled **350×350** layers (1 px = 1 m), generated by import tool
- `import_report.json` — generated by `tools/import_semantic_map.gd`

## Import

```bash
godot --headless --path . --script tools/import_semantic_map.gd
```

Smoke test:

```bash
godot --headless --path . --script tests/smoke/test_semantic_map_import.gd
godot --headless --path . --script tests/smoke/test_grid_compiler.gd
```
